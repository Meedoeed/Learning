# Отчеты по обучению

Репозиторий для хранения ежедневных отчетов по направлениям:

- Сетевая инженерия (Cisco, VLAN, маршрутизация, ACL)
- Linux (ARP, iproute2, nmcli, NetworkManager)
- Сети по Олиферу (Ethernet, IP, маски, CIDR)
- DevOps (Ansible, Docker, модель зрелости)
- Индивидуальное задание (SSH Sync Service на Go)

Навигация по темам

Сетевая инженерия (Cisco)
- VLAN, Trunk, Access
- Router-on-a-stick
- ACL (access-lists)
- STP, Native VLAN
- Проект: офис с изоляцией отделов

Linux
- ARP-таблица: просмотр, очистка, статические записи, спуфинг
- /proc/net/dev и сетевая статистика
- nmcli: статический IP, решение проблемы unmanaged
- MTU и jumbo-кадры
- Маршруты: временные и постоянные

Сети по Олиферу
- Token Ring, FDDI
- Разделяемая vs коммутируемая среда
- CSMA/CD (почему устарел)
- IP-адресация, классы, маски

DevOps
- Модель зрелости (CMMI)

Индивидуальное задание (SSH Sync Service)
- Проблематика и цели
- Roadmap
- Стек: Go, echo, zerolog, Prometheus, Postgres, React
- Архитектура: слоистая (config, domain, repository, service, handler)
- Текущий этап: config + logger

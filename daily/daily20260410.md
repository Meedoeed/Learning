# Решение задачи из праткикума

Написал такой докерфайл для своего приложения

``` dockerfile

FROM golang:1.25.2-alpine AS builder

WORKDIR /app

COPY go.mod go.sum ./
RUN go mod download

COPY . .

RUN CGO_ENABLED=0 GOOS=linux go build -ldflags="-w -s" -o deaddrop ./cmd/api

FROM alpine:latest

RUN apk --no-cache add ca-certificates tzdata
RUN adduser -D -H myuser

WORKDIR /app

COPY --from=builder /app/deaddrop .

USER myuser
EXPOSE 8080

CMD ["./deaddrop"]
```

``` powershell
docker build -t my-go-app .
```
после запуска все этапы сборки докер-контейнера были пройдены

Запустил контейнер

``` powershell
docker run -d -p 8080:8080 --name deaddrop-app my-go-app
```
